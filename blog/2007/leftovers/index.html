Leftovers

Aug 28, 2007


There is no start,
there is no end.
But, there was a journey
which never end.

There is no sun,
there is no shade.
But, there was a colour
which also fade.

There is no sorrow,
there is no grief.
But, there was a tear
though very brief.

There is no parting,
there is no distance.
But, there was a closeness
lasted for an instance.

There is no me,
there is no you.
But, there was a us
broken somehow.
