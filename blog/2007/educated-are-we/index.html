Educated, Are we?

May 29, 2007

I was having a little debate with a so called friend of mine about turning off computer while leaving from office.

This was the argument he presented.


  1. It takes too much time (2 mins i guess) to turn on the computer.
  2. I use to many applications which require logging in (I say max 10 apps, which can be automated using a simple script). So this takes lot of time.
  3. And I want to start working as soon as i hit the office without wasting time.


I will say it would be taking around 5-10 mins for applications logging in, comp booting up. If Linux would take lesser may be.
So we high-tech professionals are ready to waste electricity, which is already scarce in countries like ours so that we can save 5-10 mins
and utilize those 5-10 mins instead on using useless stuff like Orkut.

Really we are highly educated people now!!<div class="blogger-post-footer"></div>
